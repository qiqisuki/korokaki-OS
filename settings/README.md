# 设置模块

**Phase 2.5：配置面板**

## 职责
- API Key 配置（加密存储）
- 浮窗设置：透明度、位置、大小、置顶开关
- 提醒频率：提醒间隔、免打扰时段
- 音乐设置：网易云 API 地址、默认音量
- 角色设置：称呼偏好、性格强度

## 实现
- 托盘菜单 → "设置" → 弹出设置窗口
- electron-store 加密存储敏感信息（API Key）
- 设置变更通过 IPC 实时通知浮窗重载

## 配置文件
- `~/.desktop-mascot/settings.json` — 用户配置
- API Key 单独存储在系统密钥链（keytar）或加密文件
