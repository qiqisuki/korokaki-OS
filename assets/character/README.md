# 角色素材清单

> 在画师/素材到位之前，Phase 1 用纯 CSS 几何小人顶上。

## 素材需求（按优先级）

| 素材 | 状态 | 来源方案 |
|---|---|---|
| idle 默认姿态 | ❌ 未准备 | CSS 几何小人（临时）/ AI 生成 / 画师约稿 |
| blink 眨眼帧 × 2 | ❌ 未准备 | CSS 动画（临时，可用 @keyframes 模拟） |
| happy 开心 | ❌ 未准备 | 同上 |
| angry 生气 | ❌ 未准备 | 同上 |
| blush 害羞 | ❌ 未准备 | 同上 |
| sleep 休眠 | ❌ 未准备 | 同上 |
| 应用图标 .ico/.icns | ❌ 未准备 | AI 生成 / 简易 SVG |

## 临时方案（CSS 几何小人）

不依赖任何外部素材，纯 CSS 画一个：
- 圆形头 + 三角形身体 + 两点眼睛
- 眨眼用 `transform: scaleY(0)` + `animation`
- 表情切换用 CSS class 控制

## 升级路径

1. CSS 小人 → MVP 先用
2. 找素材/AI 生成 → 替换为 PNG sprite
3. 画师约稿 → 最终版本
4. Live2D 骨骼 → 长期目标

## 尺寸规格（最终版本）

- 立绘：512 × 512 PNG，透明背景
- 眨眼帧：512 × 512 × 2 帧
- 应用图标：256 × 256 (Windows), 512 × 512 (Mac)
